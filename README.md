# CRMProject (Data Analysis)

Quick setup:

```bash
# Activate the project's virtual environment
source .venv/bin/activate

# Install pinned deps (already created)
pip install -r requirements.txt

# Start Jupyter Lab
jupyter lab
```

Notes:
- A `notebooks/verify_env.ipynb` notebook is included to verify installed packages and as a starter template for A/B testing workflows.
- Add your data to `data/` (create folder) and follow the notebook sections for EDA and testing.
